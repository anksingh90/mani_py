# Programming Practice Questions

## 1. Input two numbers and display the larger / smaller number
Write a program that takes two numbers as input from the user and displays which number is larger and which is smaller.

## 2. Input three numbers and display the largest / smallest number
Write a program that takes three numbers as input from the user and displays the largest and smallest among them.

## 3. Generate the following patterns using nested loop

### a) Right-angled triangle pattern with asterisks
```
*
**
***
****
*****
```

### b) Inverted number triangle pattern
```
1 2 3 4 5
1 2 3 4
1 2 3
1 2
1
```

### c) Alphabet triangle pattern
```
A
AB
ABC
ABCD
ABCDE
```

## 4. Write a program to input the value of x and n and print the sum of the following series

### a) Series: 1 + x + x² + x³ + x⁴ + ... + xⁿ
Calculate and display the sum of this geometric series.

### b) Series: 1 - x + x² - x³ + x⁴ - ... + xⁿ
Calculate and display the sum of this alternating series.

## 5. Determine whether a number is a perfect number, a palindrome
Write a program to check if a given number is:
- A perfect number (sum of divisors equals the number)
- A palindrome (reads the same forwards and backwards)

## 6. Display the terms of a Fibonacci series
Write a program to generate and display the Fibonacci sequence up to n terms or up to a specified value.

## 7. Count and display the number of vowels, consonants, uppercase, lowercase characters in string
Write a program that analyzes a string and counts:
- Number of vowels
- Number of consonants
- Number of uppercase characters
- Number of lowercase characters

## 8. Input a list of numbers and find the smallest and largest number from the list using loop
Write a program that takes a list of numbers as input and finds the minimum and maximum values using loops.

## 9. Create a dictionary with the roll number, name and marks of n students in a class and display the names of students who have scored marks above 75
Write a program to:
- Create a dictionary storing student information (roll number, name, marks)
- Display names of students who scored more than 75 marks

## 10. Write a program to take input from the user and print reverse order
Write a program that takes input (string, number, or list) from the user and displays it in reverse order.
